## Description

Aplicação de acesso aos dados da ADS - Agência de Desenvolvimento Sustentável do Amazonas

## Instalação

```bash
$ npm install
```

## Executando o Aplicativo

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Testes

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
